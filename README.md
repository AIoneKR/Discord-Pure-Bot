# ko-KR
### Pure Bot은 관리, 게임, 채팅을 할수있습니다.
### 한국어만을 지원합니다.
### 봇을 소개하자면 관리, 게임, 채팅을 할수있는 깔끔한 봇입니다.
### 봇에 대한 불편함과 문의사항이 있다면 봇명령어 %문의 (문의할 내용)으로 문의 해주시기 바랍니다.
****
# 명령어
## 기본 prefix는 % 입니다
### (개인 Prefix)도움말 / 도움말을 보여줍니다.
****
# .env 파일을 추가합니다.
```js
TOKEN=(토큰)
botname=(봇이름)
v= v(버전)
prefix=(접두사)
dvr=(개발자ID)
```
